AI Stock Market Prediction System

An AI-powered stock analysis platform that combines deep learning (LSTM) and Large Language Models (LLMs) to generate stock trading signals and natural language explanations.

The system predicts BUY / HOLD / SELL actions based on historical stock data and uses DeepSeek and GPT to explain the model's prediction.

This project demonstrates a full-stack AI system integrating:

Machine Learning

LLM reasoning

Backend APIs

Frontend visualization

Database storage

System Architecture

The system contains four main layers:

Frontend (User Interface)

Backend API (FastAPI)

AI Prediction Model (LSTM)

LLM Explanation Layer (DeepSeek + GPT)

Workflow:

User (Browser)
      │
      ▼
Frontend Dashboard
(Vue.js + Chart.js)
      │
      ▼
Backend API
(FastAPI)
      │
      ├── LSTM Prediction Model
      │
      ├── SQLite Database
      │
      └── LLM Layer
           ├── DeepSeek
           └── GPT

The backend produces structured JSON output, which is then explained by LLMs.

Python Version Requirement

This project requires:

Python 3.9.6

Check your Python version:

python --version

Expected output:

Python 3.9.6
Step 1 — Clone the Repository
git clone https://github.com/YOUR_USERNAME/ai-stock-market.git
cd ai-stock-market
Step 2 — Create a Virtual Environment

Mac / Linux:

python3.9 -m venv .venv
source .venv/bin/activate

Windows:

python -m venv .venv
.venv\Scripts\activate

After activation you should see:

(.venv)

in the terminal.

Step 3 — Install Dependencies

Install required Python packages:

pip install -r requirements.txt

Main libraries used in this project:

fastapi
uvicorn
tensorflow
pandas
numpy
yfinance
joblib
python-dotenv
openai
sqlite3
Step 4 — Configure API Keys

Create a .env file in the project root.

Example:

DEEPSEEK_API_KEY=your_deepseek_api_key
OPENAI_API_KEY=your_openai_api_key

These keys are required for generating explanations using LLMs.

Step 5 — Start the Backend Server

Run the FastAPI backend using Uvicorn:

python -m uvicorn api.server:app --reload

If successful you will see:

Uvicorn running on http://127.0.0.1:8000
Step 6 — Open the Frontend Dashboard

Open the dashboard page in your browser:

frontend/index.html

or visit:

http://127.0.0.1:8000

The dashboard allows users to:

enter stock tickers

view prediction results

visualize price trends

compare LLM explanations

Step 7 — Test the API

Example API endpoint:

http://127.0.0.1:8000/compare?ticker=NVDA

Example response:

{
  "ticker": "NVDA",
  "action": "HOLD",
  "probabilities": {
    "SELL": 0.0833,
    "HOLD": 0.5745,
    "BUY": 0.3422
  }
}
Database

The project uses SQLite for storing stock price data.

Database location:

data/stock.db

Table:

stock_prices

Structure:

ticker
date
open
high
low
close
volume

If a ticker does not exist in the database, the system automatically downloads data using Yahoo Finance API and stores it.

Machine Learning Model

The prediction model is an LSTM neural network trained on historical stock market data.

Input features:

open
high
low
close
volume
ret1
ma5
ma20
vol10
mom10

Prediction horizon:

20 trading days

Output classes:

SELL
HOLD
BUY

The model outputs probabilities for each action.

LLM Explanation Layer

The system integrates two LLMs:

DeepSeek
OpenAI GPT

Both models receive the prediction JSON and generate explanations including:

suggestion

reasoning

risk warning

confidence level

This allows comparison between different LLM interpretations.
